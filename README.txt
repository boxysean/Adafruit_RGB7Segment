This is a library for the Adafruit RGB 7 Segment with TLC5947 breakout board

Written by Sean McIntyre.
MIT license, all text above must be included in any redistribution

Place the Adafruit_RGB7Segment library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE

**** Also requires the Tlc5940 library for Arduino. ****
Download it from here:
	https://code.google.com/p/tlc5940arduino/
